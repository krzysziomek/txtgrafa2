# Pliki kategorii: Overlay

Wrzuć tutaj wszystkie pliki `.zip` z overlayami.
Manifest (`public/manifest.json`) jest generowany automatycznie przy `vite build`
oraz przy starcie dev-servera, więc nowo dodane pliki pojawią się na stronie
po przebudowie / odświeżeniu.

Format nazw — dowolny. Kody kolorów typu `§b`, `§a`, `§f` są strippowane
z nazwy wyświetlanej na stronie. Plik trzymany jest pod oryginalną nazwą
(URL-encoded).

Plik z najnowszą datą modyfikacji pojawia się jako pierwszy.
